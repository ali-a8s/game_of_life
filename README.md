# Game of Life

## Rules of the Game
- A live cell with 2 or 3 live neighbors stays alive.
- A dead cell with exactly 3 live neighbors becomes alive.
- All other live cells die.  
  (A live cell with fewer than 2 neighbors dies of loneliness;  
   a live cell with more than 3 neighbors dies of overcrowding.)

## Features
- Set the width and height of the grid
- Set the speed of each generation (in seconds)
- Choose the initial percentage of live cells (0–100)
- Shows the generation number and live cell count each step
